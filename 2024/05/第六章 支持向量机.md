# 第六章 支持向量机

在机器学习中的支持向量机（SVM）算法里，**间隔**和**支持向量**是两个关键概念。下面详细解释它们的含义和重要性。

### 间隔（Margin）

**间隔**是指支持向量机中分隔超平面（即分类器）与最近的训练样本之间的距离。在支持向量机的优化过程中，我们希望最大化这个间隔。间隔越大，分类器的鲁棒性越强，泛化能力也通常越好。间隔可以分为两种：

**硬间隔（Hard Margin）**：这是在没有任何误分类的情况下最大化间隔。硬间隔支持向量机要求数据完全线性可分，即能够找到一个超平面，将所有样本完美分隔开。此时，我们优化的问题是：

$ \max_{\mathbf{w}, b} \frac{2}{\|\mathbf{w}\|} $

**软间隔（Soft Margin）**：这是允许一些误分类存在的情况下最大化间隔。软间隔支持向量机引入了松弛变量 \(\xi_i\)，允许某些样本违反间隔要求。优化问题变成了在最大化间隔和最小化误分类之间找到一个平衡。对应的优化问题为：

$\min_{\mathbf{w}, b, \xi} \frac{1}{2} \|\mathbf{w}\|^2 + C \sum_{i=1}^{n} \xi_i$

其中，\(C\) 是一个正则化参数，用于控制间隔和误分类之间的权衡。

### 支持向量（Support Vectors）

**支持向量**是指位于决策边界上或违反间隔条件的训练样本。具体来说，它们是距离分隔超平面最近的点，直接影响到分类器的构建。这些点是训练数据中的重要子集，SVM 的决策函数完全由这些点确定。

- 在硬间隔情况下，支持向量是正好位于分隔超平面的间隔边界上的点。
- 在软间隔情况下，支持向量不仅包括位于间隔边界上的点，还包括那些位于错误分类区域的点（即违反间隔约束的点）。

支持向量的存在对于支持向量机的决策面具有决定性作用。具体来说，给定训练数据 \(($\mathbf{x}_i, y_i)$)，SVM 的决策函数可以表示为：

$f(\mathbf{x}) = \text{sign}(\sum_{i \in SV} \alpha_i y_i K(\mathbf{x}_i, \mathbf{x}) + b)$

其中，\($\alpha_i\ $) 是拉格朗日乘子， \($K(\mathbf{x}_i, \mathbf{x})$) 是核函数， \(SV\) 表示支持向量的集合。

### 例子和图示

**图示解释**：

- 分隔超平面：通常是一条直线（在二维）或一个平面（在三维）将两类样本分开。

- 支持向量：在二维平面上，它们是离分隔超平面最近的点。通常在图中用特殊标记表示（如圈或星号）。

- 间隔：从分隔超平面到最近的支持向量的垂直距离。

**数学表达**：

- 假设分隔超平面为 \($\mathbf{w} \cdot \mathbf{x} + b = 0$\)。
- 间隔 \(\gamma\) 定义为：

$ \gamma = \frac{1}{\|\mathbf{w}\|} $
